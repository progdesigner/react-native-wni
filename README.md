# react-native-wni

React Native WebView에서 네이티브 디바이스/앱/OS 정보 및 명령 브릿지를 제공합니다.

## 설치

```sh
npm install react-native-wni
```

## 빌드 (패키지 개발자용)

```sh
npm run build
```

## 사용법

```ts
import WNInterface from 'react-native-wni';
// WNInterface 사용 예시
```

## 패키지 정보
- main: dist/index.js
- types: dist/index.d.ts
